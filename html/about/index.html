About

The Improved TSA Guide was created in April, 2021 by students at Mountlake Terrace High School as a practical, community-run guide for new Technology Student Association (TSA) members. That's pretty much it; this project is very early in development.


How can I contribute?


This project is not endorsed by TSA; it needs volunteers to keep it running. There are many ways you can contribute.

  • Update information
  • Add new event information
  • Find and fix bugs or grammatical errors
  • Add new stories or updates for the news section
  • Add images

To get started, visit the Github repository. For more major updates like redesigning the site, contact the owner of this repository.
