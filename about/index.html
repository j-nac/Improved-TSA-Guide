About

The Improved TSA Guide was created in April, 2021 by students in the Edmonds School District as a guide for new Technology Student Association (TSA) members. It is based on a similar project by Github users j-nac and therealanby which is now private. That's pretty much it; this project is very early in development.


How can I contribute?


This project is not endorsed the Technology Student Association. We need volunteers to keep it running. There are many ways you can contribute including:

  • Updating information
  • Adding new event information
  • Finding and fixing bugs or grammatical errors
  • Adding new stories or updates for the news section
  • Adding images

To get started, visit the Github repository where you can add issues, fix issues, and join the discussion. For more major updates like redesigning the site, contact the maintainer of this repository.


Won't This Benefit Rival Competition?


Probably, but that's ok. At the end of the day, TSA is about learning; learning about career opportunities in STEM, learning to improve yourself, learning to be a better leader. We believe information should be free and we hope everyone takes something useful from this project.
